# UdpServer
基于Udp通信的简单翻译服务器。

## 原理
通过读取当前目录下的dict.txt，构建哈希表，接受英文单词并应答对应的中文。若不存在对应的单词，将回复 Unknow 。

本项目使用了C++17的语言特性。

## 技术亮点
1. 在使用分层封装Udp服务时，只需要一个端口作为参数，就可以创建一个Udp服务。
2. 清晰的日志，标注了时间和日志对应的文件名和行号，方便debug，优化使用方式，支持像cout一样输出。

## 使用方式
使用make进行编译
```bash
make
```

快捷删除编译出来的可执行文件
```bash
make clean
```

无中间文件
